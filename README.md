# Quebra-Multas — Site Oficial

Este é o repositório oficial do site da **Quebra-Multas**, uma empresa especializada na defesa de motoristas contra penalidades de trânsito indevidas.

---

## 📄 Estrutura do site

O site é composto por páginas estáticas em HTML, com foco em carregamento rápido, compatibilidade com campanhas do Google Ads e boa experiência para o usuário.

/
├── index.html # Página principal
├── bafometro.html # Página específica sobre bafômetro
├── suspensao.html # Página de suspensão de CNH
├── contato.html # Página de contato
├── ppd.html # Página de permissão para dirigir
└── assets/ # Imagens, CSS e scripts
├── style.css
├── script.js
└── images/

---

## 🚀 Deploy e hospedagem

O site está atualmente publicado via **Hostinger**, e as páginas estão diretamente ligadas às campanhas de tráfego pago (Google Ads), com foco em conversão por WhatsApp.

---

## 🤖 Uso com Codex (ChatGPT)

Este repositório está preparado para utilização com o **Codex do ChatGPT**, permitindo:
- Análise e correção de código;
- Otimizações de performance e SEO;
- Adição de novos recursos HTML/CSS/JS;
- Refatoração de estrutura.
